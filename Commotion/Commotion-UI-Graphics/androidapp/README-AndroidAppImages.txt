Notes on Android App Graphics:

Master SVG files contain all layers used to generate the PNG images here. To produce proper scaling support on Android devices with different screen sizes and resolutions, each graphic element is saved in multiple formats and sizes per Android's Developer documentation:

http://developer.android.com/design/style/iconography.html
http://developer.android.com/design/style/devices-displays.html
http://developer.android.com/guide/practices/screens_support.html

xhdpi	~320dpi
hdpi	~240dpi
mdpi	~160dpi
ldpi	~120dpi
